# TP 1 : Création d'une application React simple (Composants et Props)
Ce TP vous a permis de mettre en pratique concrètement les compétences fondamentales suivantes :

Initialisation d’un projet React.

Création et utilisation de composants fonctionnels avec JSX.

Transmission et validation de données entre composants via les props et PropTypes.

Structuration efficace d’une application React.
